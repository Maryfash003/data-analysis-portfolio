#Sales Performance Analysis & Dashboard

## Overview
This project analyzes sales performance data using **Excel** and **Tableau**.  
It demonstrates skills in data cleaning, visualization, and storytelling with data.  
The goal is to provide actionable insights into revenue trends, product performance, and regional sales.


## Tools & Techniques
- **Excel**: Data cleaning, pivot tables, trend analysis  
- **Tableau**: Interactive dashboards, charts, and KPIs  
- **Data Visualization**: Highlighting key metrics such as revenue growth, top products, and regional comparisons  


## Files Included
- `sales_analysis.xlsx` → Raw data and Excel-based analysis  
- `sales_dashboard.twbx` → Tableau workbook with interactive dashboard  
- `dashboard_screenshot.png` → Preview image of the dashboard  


## Key Insights
- Identified top-performing products and regions driving revenue growth  
- Highlighted seasonal trends and monthly performance variations  
- Built KPIs to track sales targets vs. actuals
